Triprex - Travel Agency and Tour Booking Laravel Script.

TripRex is an all-in-one, feature-packed Laravel script designed for travel agencies, tour operators, and booking businesses. From tours and hotels to visa processing, transport rentals, and local experiences – TripRex handles it all with ease and efficiency.

Highlights Feature

  • Tour Management: Create, manage, and book tours.
  • Hotel: Manage hotel listings and reservations.
  • Transports Services: Handle flights, transport, and itinerary planning.
  • Visa Assistance: Provide visa application tracking and support.
  • User Management: Customers, Agents and admin roles with authentication.
  • Payment Integration: Support for multiple payment gateways.
  • Multi-Language & Currency Support: Localization features.
  • RTL Supported
  • Review & Ratings: Customers can leave feedback on tours and hotels.
  • Deposit: Customers can deposit balance.
  • Withdrawal: Merchant can withdraw her balance.
  • Notifications: Email and SMS notifications for bookings.
  • Booking Management
  • Blog System
  • Testimonial
  • Understandable Documentation & Easy Installation Process
  • Custom Page Builder so that easily anyone can customize by Drag & Drop
  • Menu Builder
  • Safe and Secure Transaction
  • Support Ticket System
  • Well Designed and Decorated Frontend Design

Installation

Requirements

Before installing our Tour & Travel application, ensure that your server meets the following requirements:

  • Apache, nginx, or another compatible web server
  • PHP >= 8.2 or higher
  • Minimum MySQL version: 5.7
  • MySQL Database server
  • PDO PHP extension
  • OpenSSL PHP extension
  • mbstring PHP extension
  • exif PHP extension
  • fileinfo PHP extension
  • xml PHP extension
  • Ctype PHP extension
  • JSON PHP extension
  • Tokenizer PHP extension
  • cURL PHP extension
  • Ensure the mod_rewrite Apache module is enabled

SSL

Install SSL certificate on your hosting/VPS

You need to install an SSL certificate on your hosting/VPS first. You can purchase an SSL certificate or use free SSL, for example Let's Encrypt.

Redirect http to https

  • Change the following code in .env file from APP_URL=http://domain.com to APP_URL=https://domain.com.

  • There have 3 options:

  • Option 1: Add ENABLE_HTTPS_SUPPORT=true to .env file

  • Option 2: Add to .env file following code:

          FORCE_SCHEMA=https
            FORCE_ROOT_URL=https://domain.com
            ENABLE_HTTPS_SUPPORT=false
  • Option 3: If you are using hosting and cPanel, you can Force HTTPS Redirection for your domains. Check this article.

WARNING

If it doesn't work, you have to config it in .htaccess or Nginx config.
